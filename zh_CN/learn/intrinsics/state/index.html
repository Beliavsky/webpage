associated#

名称#

associated(3) - [STATE] 指针或指针/目标对的状态

语法#

result = associated(pointer, target)

说明#

associated(pointer [, target]) 确定指针pointer 的状态，或者pointer 是否与目标target 相关联。

参数#

  • pointer

    pointer 应具有 pointer 属性，它可以是任何类型。

  • target

    （可选）target 应为指针或目标。它必须具有与pointer相同的类型、种类类型参数和数组维度数。

pointer 和 target 的关联状态都不能定义。

返回#

associated(pointer) 返回一个 logical 类型的标量值。有几种情况：

  1. 当可选的 target 不存在时，如果 pointer 与目标相关联，则 associated(pointer) 为真；否则，它返回 false。

  2. 如果 target 存在并且是标量, 如果 target 不是零大小的存储序列并且与 pointer 关联的目标占用相同的存储单元，则结果为真. 如果 pointer 未关联, 则结果为假.

  3. 如果target存在并且是数组，如果target和pointer形状相同，不是零大小数组，并且是元素不是零大小存储序列的数组，并且target和pointer在数组元素顺序上占据相同的存储单元，则结果为真。

    与第二种情况一样，如果指针未关联，则结果为假。

  4. 如果target存在并且是标指针，如果target关联了pointer，则与target关联的目标不是零大小的存储序列，并且占用相同的存储单元，则结果为真。

    如果取消关联target或pointer，则结果为**.false.**。

  5. 如果target存在并且是数组指针，则如果pointer关联的目标与target关联的目标形状相同，且不是零大小数组，且数组元素不是零大小存储序列，且target和pointer在数组元素顺序上占用相同的存储单元，则结果为真。如果解除关联target或pointer，则结果为假。

示例#

示例程序：

program demo_associated
implicit none
real, target  :: tgt(2) = [1., 2.]
real, pointer :: ptr(:)
   ptr => tgt
   if (associated(ptr)     .eqv. .false.) &
   & stop 'POINTER NOT ASSOCIATED'
   if (associated(ptr,tgt) .eqv. .false.) &
   & stop 'POINTER NOT ASSOCIATED TO TARGET'
end program demo_associated

extends_type_of#

名称#

extends_type_of(3) - [状态] 确定a的动态类型是否为mold的动态类型的扩展。

语法#

result=extends_type_of(a, mold)

说明#

extends_type_of(3) 为 .true. 当且仅当 a 的动态类型是 mold动态类型的扩展.

选项#

  • a

    应为一个扩展类型的对吸纳更. 如果是指针, 则它不应该处于未关联状态.

  • mold

    应为一个扩展类型的对吸纳更. 如果是指针, 则它不应该处于未关联状态.

返回#

  • 结果

    默认为逻辑标量。

  • value

    如果mold是无限多态的，并且是一个解除关联的指针或未分配的可分配变量，则结果为真；否则，如果a是无限多态的，并且是一个解除关联的指针或未分配的可分配变量，则结果为假；否则，当且仅当a的动态类型是mold的动态类型的扩展类型时，结果为真。

    取消关联的指针或未分配的可分配变量的动态类型是其声明类型。

is_iostat_end#

名称#

is_iostat_end(3) - [状态] 测试EOF值

语法#

function is_iostat_end(i)

    logical function   :: is_iostat_end (i) result(yesno)
    integer,intent(in) :: i

说明#

is_iostat_end(3) 测试一个变量 (假设作为 I/O 语句的返回状态) 是否具有 “end of file” I/O 状态值.

该函数等同于与内置模块 iso_fortran_env的 iostat_end 参数的值相比较.

参数#

  • i

    一个 测试是否指示文件结尾 integer 状态值.

返回#

返回默认类型的_logical_，如果i具有指示**iostat=**说明符的文件结束条件的值则为 .true.，否则为 .false..。

示例#

示例程序：

program demo_iostat
implicit none
real               :: value
integer            :: ios
character(len=256) :: message
   write(*,*)'Begin entering numeric values, one per line'
   do
      read(*,*,iostat=ios,iomsg=message)value
      if(ios.eq.0)then
         write(*,*)'VALUE=',value
      elseif( is_iostat_end(ios) ) then
         stop 'end of file. Goodbye!'
      else
         write(*,*)'ERROR:',ios,trim(message)
      endif
      !
   enddo
end program demo_iostat

is_iostat_eor#

名称#

is_iostat_eor(3) - [状态] 测试记录结束值

语法#

result = is_iostat_eor(i)

说明#

is_iostat_eor 测试变量是否具有I/O状态“end of record”的值。该函数相当于将该变量与内置模块iso_fortran_env的iostat_eor参数进行比较。

参数#

  • i

    类型应为 integer.

返回#

返回默认种类的 logical, 如果i具有指示iostat=说明符的文件结束条件的值它为.true.。否则为.false.。

示例#

示例程序：

program demo_is_iostat_eor
implicit none
integer :: stat, i(50)

  open(88, file='test.dat', form='unformatted')
  read(88, iostat=stat) i

  if(is_iostat_eor(stat)) stop 'end of record'

end program demo_is_iostat_eor

move_alloc#

名称#

move_alloc(3) - [] 将分配从一个对象移动到另一个对象

语法#

call move_alloc(src, dest)

说明#

move_alloc(src, dest) 将分配从 src 移动到 *dest。src 将在此过程中解除分配。

参数#

  • src

    可分配的，intent(inout)，可以是任何类型和种类。

  • dest

    可分配的，intent(out)，应与 src 具有相同的类型、种类和等级。

示例#

分配更大网格的基本示例程序

program demo_move_alloc
implicit none
! Example to allocate a bigger GRID
real, allocatable :: grid(:), tempgrid(:)
integer :: n, i

   ! initialize small GRID
   n = 3
   allocate (grid(1:n))
   grid = [ (real (i), i=1,n) ]

   ! initialize TEMPGRID which will be used to replace GRID
   allocate (tempgrid(1:2*n))    ! Allocate bigger grid
   tempgrid(::2)  = grid         ! Distribute values to new locations
   tempgrid(2::2) = grid + 0.5   ! initialize other values

   ! move TEMPGRID to GRID
   call MOVE_ALLOC (from=tempgrid, to=grid)

   ! TEMPGRID should no longer be allocated
   ! and GRID should be the size TEMPGRID was
   if (size (grid) /= 2*n .or. allocated (tempgrid)) then
      print *, "Failure in move_alloc!"
   endif
   print *, allocated(grid), allocated(tempgrid)
   print '(99f8.3)', grid
end program demo_move_alloc

结果：

    T F
      1.000   1.500   2.000   2.500   3.000   3.500

present#

名称#

present(3) - [STATE] 确定是否指定了可选的虚拟参数

语法#

result = present(a)

   function present (a)
   logical :: present

说明#

确定是否存在可选的虚拟参数。

参数#

  • a

    可以是任何类型，可以是指针、标量或数组值，或虚拟过程。它应该是在当前子例程或函数中可访问的可选虚拟参数的名称。

返回#

如果存在可选参数 a，则返回 .true.，否则返回 .false.。

示例#

示例程序：

program demo_present
implicit none
   write(*,*) func(), func(42)
contains

integer function func(x)
integer, intent(in), optional :: x
   if(present(x))then
     func=x**2
   else
     func=0
   endif
end function

end program demo_present

结果：

     0        1764

same_type_as#

名称#

same_type_as(3) - [STATE] 查询动态类型是否相等**

语法#

result = same_type_as(a, b)

说明#

查询动态类型是否相等。

参数#

  • a

    应为可扩展声明类型或无限多态的对象。

  • b

    应为可扩展声明类型或无限多态的对象。

返回#

返回值是默认逻辑类型的标量。当且仅当 a 的动态类型与 b 的动态类型相同时，它才是真的。
