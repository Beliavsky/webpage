有关用于构建的编译器和编译器选项的信息#

编译器选项#

名称#

compiler_options(3) - [COMPILER INQUIRY] 传递给编译器的选项

语法#

str = compiler_options()

说明#

compiler_options 返回一个带有用于编译的选项的字符串。

参数#

无。

返回#

返回值是具有系统相关长度的默认类型字符串。它包含用于编译文件的编译器标志，称为 compiler_options 内在函数。

示例#

示例程序：

program demo_compiler_version
use, intrinsic :: iso_fortran_env, only : compiler_version, compiler_options
implicit none
   print '(4a)', &
      'This file was compiled by ', &
      compiler_version(),           &
      ' using the options ',        &
      compiler_options()
end program demo_compiler_version

结果：

   This file was compiled by GCC version 5.4.0 using the options
   -I /usr/include/w32api -I /home/urbanjs/V600/lib/CYGWIN64_GFORTRAN
   -mtune=generic -march=x86-64 -g -Wunused -Wuninitialized -Wall
   -std=f2008 -fbounds-check -fbacktrace -finit-real=nan
   -fno-range-check -frecord-marker=4
   -J /home/urbanjs/V600/lib/CYGWIN64_GFORTRAN

标准#

Fortran 2008

另见#

compiler_version(3),,iso_fortran_env(7)

compiler_version#

名称#

compiler_version(3) - [COMPILER INQUIRY] 编译器版本字符串

语法#

str = compiler_version()

说明#

compiler_version(3) 返回一个包含编译器名称和版本的字符串。

参数#

无。

返回#

返回值是具有系统相关长度的默认类型字符串。它包含编译器的名称及其版本号。

示例#

示例程序：

program demo_compiler_version
use, intrinsic :: iso_fortran_env, only : compiler_version, compiler_options
implicit none
   print '(4a)', &
      'This file was compiled by ', &
      compiler_version(),           &
      ' using the options ',        &
      compiler_options()
end program demo_compiler_version

结果：

   This file was compiled by GCC version 5.4.0 using the options
   -I /usr/include/w32api -I /home/urbanjs/V600/lib/CYGWIN64_GFORTRAN
   -mtune=generic -march=x86-64 -g -Wunused -Wuninitialized -Wall
   -std=f2008 -fbounds-check -fbacktrace -finit-real=nan
   -fno-range-check -frecord-marker=4
   -J /home/urbanjs/V600/lib/CYGWIN64_GFORTRAN

标准#

Fortran 2008

另见#

compiler_options(3),,iso_fortran_env(7)
