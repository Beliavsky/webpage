co_broadcast#

名称#

co_broadcast(3) - [COLLECTIVE] 将值从当前镜像复制到所有镜像

语法#

call co_broadcast(a, source_image, stat, errmsg)

说明#

co_broadcast(3) 将镜像索引为 source_image 的上的参数 a 的值复制到当前组中的所有镜像。 a 被定义为内部赋值。如果执行成功并且存在 stat，则将其赋值为零。如果执行失败，stat 被分配一个非零值，如果存在，errmsg 被分配一个值来描述发生的错误。

参数#

  • a

    intent(inout) 参数；在当前组的所有镜像上应具有相同的动态类型和类型参数。如果它是一个数组，它应该在所有镜像上具有相同的形状。

  • source_image

    一个标量整数表达式。它在所有镜像上应具有相同的值，并引用当前组的镜像。

  • stat

    （可选）标量整数变量

  • errmsg

    （可选）一个标量字符变量

示例#

示例程序：

program demo_co_broadcast
implicit none
integer :: val(3)
   if (this_image() == 1) then
      val = [1, 5, 3]
   endif
   call co_broadcast (val, source_image=1)
   print *, this_image(), ":", val
end program demo_co_broadcast

co_lbound#

名称#

co_lbound(3) - [COLLECTIVE] 数组的下维数边界

语法#

result = co_lbound(coarray, dim, kind)

说明#

返回coarray的下界，或沿dim单维的单个下界。

参数#

  • 数组

    应为任何类型的coarray。

  • 暗淡

    （可选）应为标量 _ 整数 _。

  • 种类

    （可选）一个 integer 初始化表达式，指示结果的种类参数。

返回值#

返回值的类型为_integer_，种类为kind。如果没有指定kind，则返回值为默认的整型。如果没有指定dim，则结果是corray的下限数组。如果指定了dim，则结果是对应于数组沿该维的下限的标量。

co_max#

名称#

co_max(3) - [集合] 当前镜像集上的最大值

语法#

call co_max(a, result_image, stat, errmsg)

说明#

co_max 确定当前编队所有镜像的 a 元素的最大值。如果 result_image 存在，则只在指定的镜像上返回 a 中的最大值，而在其它镜像上的 a 的值变为未定义。如果 result_image 不存在，则在所有镜像上返回值。如果执行成功并且存在 stat，则将其赋值为零。如果执行失败，则为 stat 指定一个非零值，如果 errmsg 存在，则为其分配一个描述发生的错误的值。

参数#

  • a

    应为整数，实数或字符变量，在并行的所有镜像上具有相同的类型和类型参数。

  • result_image

    (可选)一个标量整型表达式；如果存在，则在所有镜像上具有相同的值并指为并行中的一个镜像。

  • stat

    （可选）标量整数变量

  • errmsg

    （可选）一个标量字符变量

示例#

示例程序：

program demo_co_max
implicit none
integer :: val
   val = this_image()
   call co_max(val, result_image=1)
   if (this_image() == 1) then
     write(*,*) "Maximal value", val  ! prints num_images()
   endif
end program demo_co_max

结果：

    Maximal value           2

co_reduce#

名称#

co_reduce(3) - [集合] 缩减当前镜像集上的值

语法#

call co_reduce(a, operation, result_image, stat, errmsg)

说明#

co_reduce 确定当前编队所有镜像上 a 的值在元素方面的减少量。作为 operation 传递的纯函数用于通过传递不同镜像的 a 的值或作为参数传递这种缩减的结果值来成对地缩减 a 的值。如果 a 是一个数组，则按元素进行缩减。如果 result_image 存在，则只在指定的镜像上返回 a中的结果值，而在其它镜像上的 a 的值变为未定义。如果 result_image 不存在，则在所有镜像上返回值。如果执行成功并且存在 stat，则将其赋值为零。如果执行失败，则为 stat 分配一个非零值，如果 errmsg 存在，则为其分配一个描述发生的错误的值。

参数#

  • a

    是**intent(inout)**参数，并且应该是非多态的。如果是动态数组，则应该已经分配；如果是指针，则应该已经关联。a在并行的所有镜像上应具有相同的类型和类型参数；如果是数组，则在所有镜像上应具有相同的维度。

  • operation

    具有两个标量不可分配参数的纯函数应该是非多态的并且具有与 a 相同的类型和类型参数。该函数应返回与 a 相同类型和类型参数的不可分配标量。该函数在所有镜像上以及在数学上关于自变量的可交换性和关联性方面应相同。请注意，操作可能不是基元的，除非它是内置函数。

  • result_image

    (可选)一个标量整型表达式；如果存在，则在所有镜像上具有相同的值并指为并行中的一个镜像。

  • stat

    （可选）标量整数变量

  • errmsg

    （可选）一个标量字符变量

示例#

示例程序：

program demo_co_reduce
implicit none
integer :: val

   val = this_image()
   call co_reduce(val, myprod, 1)
   if (this_image() == 1) then
      write(*,*) "Product value", val  ! prints num_images() factorial
   endif

contains

pure function myprod(a, b)
   integer, value :: a, b
   integer :: myprod
   myprod = a * b
end function myprod

end program demo_co_reduce

Note#

虽然规则原则上允许使用内部函数，但标准的内部函数都不满足具有特定函数的标准，该特定函数接受相同类型的两个参数并返回该类型。

co_sum#

名称#

co_sum(3) - [集合] 当前镜像集上的值之和

语法#

call co_sum(a, result_image, stat, errmsg)

说明#

co_sum 累加当前团队所有镜像上 a 的每个元素的值。如果 result_image 在，则只在指定的镜像上返回 a 中的值之和，而在其它镜像上的 a 的值变为未定义。如果 result_image 不存在，则在所有镜像上返回值。如果执行成功并且存在 stat，则将其赋值为零。如果执行失败，则为 stat 分配一个非零值，如果 errmsg 存在，则为其分配一个描述发生的错误的值。

参数#

  • a

    应为整数、实数或复数变量，在编队的所有镜像上具有相同的类型和种类参数。

  • result_image

    (可选)一个标量整型表达式；如果存在，则在所有镜像上具有相同的值并指为并行中的一个镜像。

  • stat

    （可选）标量整数变量

  • errmsg

    （可选）一个标量字符变量

示例#

示例程序：

program demo_co_sum
implicit none
integer :: val
   val = this_image()
   call co_sum(val, result_image=1)
   if (this_image() == 1) then
      ! prints (n**2 + n)/2, with n = num_images()
      write(*,*) "The sum is ", val
   endif
end program demo_co_sum

结果：

    The sum is            1

event_query#

名称#

event_query(3) - [集合] 查询是否已发生coarray事件

语法#

call event_query(event, count, stat)

说明#

event_query通过调用event_wait，将已指定给event变量但尚未移除的事件数赋给count。当stat存在并且调用成功时，它会被赋值0。如果stat存在，并且调用失败，则为其赋值为正值，并为count赋值**-1**。

参数#

  • event

    (intent(in)) 在iso_fortran_env中定义的 event_type类型的标量不应编入共索引。

  • count

    (intent(out)) 精度至少为默认 integer 的标量整数。

  • stat

    (可选) 默认种类 integer 标量变量.

示例#

示例程序：

program demo_event_query
use iso_fortran_env
implicit none
type(event_type) :: event_value_has_been_set[*]
integer :: cnt
   if (this_image() == 1) then
      call event_query(event_value_has_been_set, cnt)
      if (cnt > 0) write(*,*) "Value has been set"
   elseif (this_image() == 2) then
      event post(event_value_has_been_set[1])
   endif
end program demo_event_query

num_images#

名称#

num_images(3) - [集合] 镜像的数量

语法#

result = num_images(distance, failed)

说明#

返回镜像的数量。

参数#

  • distance

    (optional, intent(in)) 非负标量整数

  • failed

    (optional, intent(in)) 标量逻辑表达式

返回值#

标量默认类型 integer。如果distance不存在或值为0，则返回当前编队的镜像数量。对于小于或等于初始编队距离的值，它返回与调用团队的距离为distance的起始编队上的镜像索引数量。如果distance大于到初始编队的距离，则返回初始团队的镜像数量。如果不存在failed，则返回镜像总数；如果值为.true.，则返回失败镜像的数量，否则返回没有失败状态的镜像的数量。

示例#

示例程序：

program demo_num_images
implicit none
integer :: value[*]
integer :: i

   value = this_image()
   sync all
   if (this_image() == 1) then
     do i = 1, num_images()
       write(*,'(2(a,i0))') 'value[', i, '] is ', value[i]
     end do
   endif

end program demo_num_images

标准#

Fortran 2008 以后. 有 DISTANCE 或 FAILED 参数, TS 18508 or later

另见#

this_image(3), image_index(3)

this_image#

名称#

this_image(3) - [COLLECTIVE] 此镜像的下标索引

语法#

result = this_image()

或

```fortran
result = this_image(distance)

或

result = this_image(coarray, dim)

说明#

返回此镜像的下标。

参数#

  • distance

    （optional，intent(in)）非负标量整数（不允许与 coarray 一起使用）。

  • coarray

    任何类型的 Coarray（optional；如果 dim 存在，则为必需）。

  • 暗淡

    默认整数标量（可选）。如果存在，dim 应介于 1 和 coarray 的 corank 之间。

返回值#

默认整数。如果 coarray 不存在，它是标量；如果 distance 不存在或值为 0，它的值是当前团队调用图像上的镜像索引，对于与初始团队的距离更小或相等的值，它返回镜像索引与调用团队距离距离的祖先团队。如果距离大于到初始团队的距离，则返回初始团队的镜像索引。否则，当 coarray 存在时，如果 dim 不存在，则返回具有 corank 元素的 rank-1 数组，其中包含指定调用镜像的 coarray 的共同下标。如果存在 dim，则返回一个标量，其中包含 this_image(coarray) 的 dim 元素的值。

示例#

示例程序：

program demo_this_image
implicit none
integer :: value[*]
integer :: i
   value = this_image()
   sync all
   if (this_image() == 1) then
      do i = 1, num_images()
         write(*,'(2(a,i0))') 'value[', i, '] is ', value[i]
      end do
   endif
end program demo_this_image

结果：

   value[1] is 1

Fortran 2008 及更高版本。使用 DISTANCE 参数，TS 18508 或更高版本

atomic_and#

名称#

atomic_and(3) - [ATOMIC:BIT MANIPULATION] 原子按位与运算

语法#

call atomic_and(atom, value, stat)

说明#

atomic_and(atom, value) 以 atom 和 value 的值之间的按位 and 原子地定义atom。当 stat 存在且调用成功时，赋值为 0。如果存在且调用失败，则赋值为正值；特别是，对于一个共索引的atom，如果远程映像已停止，则为其分配 iso_fortran_env 的 stat_stopped_image 的值，如果远程镜像失败，则为其分配 stat_failed_image 的值。
