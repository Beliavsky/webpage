一般数学函数#

acos#

名称#

acos(3) - [MATHEMATICS:TRIGONOMETRIC] 反余弦（反余弦）函数

语法#

  result = acos(x)

   TYPE(kind=KIND),elemental :: acos

   TYPE(kind=KIND,intent(in) :: x

其中 TYPE 可以是 real 或 complex，而 KIND 可以是关联类型支持的任何 KIND。

说明#

acos(x) 计算 x 的反余弦值（cos(x) 的倒数）。

参数#

  • x

    必须是 real 或 complex 类型。如果类型为_real_，则值必须满足|x| <= 1。

返回#

返回值与 x 具有相同的类型和种类。结果的 real 部分以弧度为单位，位于 0 &lt;= acos(x%re) &lt;= PI 范围内。

示例#

示例程序：

program demo_acos
use, intrinsic :: iso_fortran_env, only : real_kinds,real32,real64,real128
implicit none
character(len=*),parameter :: all='(*(g0,1x))'
real(kind=real64) :: x = 0.866_real64
real(kind=real64),parameter :: d2r=acos(-1.0_real64)/180.0_real64

    print all,'acos(',x,') is ', acos(x)
    print all,'90 degrees is ', d2r*90.0_real64, ' radians'
    print all,'180 degrees is ', d2r*180.0_real64, ' radians'
    print all,'for reference &
    &PI ~ 3.14159265358979323846264338327950288419716939937510'
    print all,'elemental',acos([-1.0,-0.5,0.0,0.50,1.0])

end program demo_acos

结果：

   acos( .8660000000000000 ) is  .5236495809318289
   90 degrees is  1.570796326794897  radians
   180 degrees is  3.141592653589793  radians
   for reference PI ~ 3.14159265358979323846264338327950288419716939937510
   elemental 3.141593 2.094395 1.570796 1.047198 .000000

acosh#

名称#

acosh(3) - [MATHEMATICS:TRIGONOMETRIC] 反双曲余弦函数

语法#

  result = acosh(x)

   TYPE(kind=KIND),elemental :: acosh

   TYPE(kind=KIND,intent(in) :: x

其中 TYPE 可以是 real 或 complex ，而 KIND 可以是相关类型支持的任何 KIND。

说明#

acosh(x) 以弧度计算 x 的反双曲余弦值。

参数#

  • x

    类型应为_real_ 或_complex_。

返回#

返回值与 x 具有相同的类型和种类。

如果 x 是 complex，则结果的虚部以弧度为单位，介于

0 &lt;= aimag(acosh(x)) &lt;= PI

示例#

示例程序：

program demo_acosh
use,intrinsic :: iso_fortran_env, only : dp=>real64,sp=>real32
implicit none
real(kind=dp), dimension(3) :: x = [ 1.0d0, 2.0d0, 3.0d0 ]
   write (*,*) acosh(x)
end program demo_acosh

结果：

 0.000000000000000E+000   1.31695789692482        1.76274717403909

asin#

名称#

asin(3) - [MATHEMATICS:TRIGONOMETRIC] 反正弦函数

语法#

result = asin(x)

    elemental TYPE(kind=KIND) function asin(x)
    TYPE(kind=KIND) :: x

其中返回值具有输入值的种类，TYPE 可能是 real 或 complex

说明#

asin(x) 计算其参数 x 的反正弦值。

反正弦是正弦函数的反函数。当已知直角三角形的斜边和对边的长度时，它通常用于三角几何学中求取角度。

参数#

  • x

    类型应为 real 和小于或等于 1 的量级；或者是_complex_。

返回#

  • 结果

    返回值与 x 具有相同的类型和种类。结果的实部以弧度为单位，位于 -PI/2 &lt;= asin(x) &lt;= PI/2 范围内。

示例#

当你知道角度对边与斜边的比率时，反正弦将允许你找到直角的度量。

因此，如果你知道火车轨道在 50 英里长的轨道上上升 1.25 英里，你可以使用反正弦来确定轨道的平均倾斜角度。给定

 sin(theta) = 1.25 miles/50 miles (opposite/hypotenuse)

示例程序：

program demo_asin
use, intrinsic :: iso_fortran_env, only : dp=>real64
implicit none
! value to convert degrees to radians
real(kind=dp),parameter :: D2R=acos(-1.0_dp)/180.0_dp
real(kind=dp)           :: angle, rise, run
character(len=*),parameter :: all='(*(g0,1x))'
  ! given sine(theta) = 1.25 miles/50 miles (opposite/hypotenuse)
  ! then taking the arcsine of both sides of the equality yields
  ! theta = arcsine(1.25 miles/50 miles) ie. arcsine(opposite/hypotenuse)
  rise=1.250_dp
  run=50.00_dp
  angle = asin(rise/run)
  print all, 'angle of incline(radians) = ', angle
  angle = angle/D2R
  print all, 'angle of incline(degrees) = ', angle

  print all, 'percent grade=',rise/run*100.0_dp
end program demo_asin

结果：

    angle of incline(radians) =    2.5002604899361139E-002
    angle of incline(degrees) =    1.4325437375665075
    percent grade=   2.5000000000000000

百分比等级是斜率，写成百分比。要计算斜率，您将上升高等除以运行路程。在示例中，50 英里的路程上升 1.25 英里，因此坡度为 1.25/50 = 0.025。写成百分比，这是 2.5 %。

对于美国来说，2% 和 1/2% 通常被认为是上限。这意味着前进 100 英尺时上升 2.5 英尺。在美国，这是美国第一条主要铁路巴尔的摩和俄亥俄州的最高等级。注意曲线会增加火车上的摩擦阻力，从而降低允许坡度。

名称#

语法#

result = asinh(x)

    elemental TYPE(kind=KIND) function asinh(x)
    TYPE(kind=KIND) :: x

其中，返回值的类型与输入值相同，类型可以是 real 或 complex

说明#

asinh(x) 计算 x 的反双曲正弦.

    类型应该是 real 或者 complex.

返回#

    write (*,*) asinh(x)

end program demo_asinh

结果：

atan#

名称#

atan(3) - [数学:三角函数] 反正切函数

语法#

   result = atan(y, x)

    TYPE(kind=KIND):: atan
    TYPE(kind=KIND,intent(in) :: x
    TYPE(kind=KIND,intent(in),optional :: y

其中，TYPE可以是_real_或_complex_，而KIND可以是关联类型支持的任何KIND。如果存在y，则x为_real`。

说明#

**atan(x)**计算 x的反正切.

参数#

  • x

    其类型应该是 real 或者 complex; 如果 y 存在, 则 x 是 _real_类型.

  • y

    应该与 x类型和种类相同. 如果 x 为零, 则y 必须不为零.

返回#

返回值与x的类型和种类相同。如果y存在，则结果与atan2(y，x)相同。否则，它是x的反正切，其中结果的实部以弧度为单位，位于**-PI/2<=atan(X)<=PI/2**的范围内

示例#

示例程序：

program demo_atan
use, intrinsic :: iso_fortran_env, only : real_kinds, &
 & real32, real64, real128
implicit none
character(len=*),parameter :: all='(*(g0,1x))'
real(kind=real64),parameter :: &
 Deg_Per_Rad = 57.2957795130823208767981548_real64
real(kind=real64) :: x
    x=2.866_real64
    print all, atan(x)

    print all, atan( 2.0d0, 2.0d0),atan( 2.0d0, 2.0d0)*Deg_Per_Rad
    print all, atan( 2.0d0,-2.0d0),atan( 2.0d0,-2.0d0)*Deg_Per_Rad
    print all, atan(-2.0d0, 2.0d0),atan(-2.0d0, 2.0d0)*Deg_Per_Rad
    print all, atan(-2.0d0,-2.0d0),atan(-2.0d0,-2.0d0)*Deg_Per_Rad

end program demo_atan

结果：

   1.235085437457879
   .7853981633974483 45.00000000000000
   2.356194490192345 135.0000000000000
   -.7853981633974483 -45.00000000000000
   -2.356194490192345 -135.0000000000000

atan2#

名称#

atan2(3) - [数学:三角函数] 反正切函数

语法#

result = atan2(y, x)

说明#

atan2(y, x) 计算复数 ( x + i y )的反正切 .

此函数可用于将笛卡尔坐标转换为极坐标，并可以确定正确象限中的角度。为了将笛卡尔坐标**(x,y)**转换为极坐标

(r,theta): $\( \begin{aligned} r &= \sqrt{x**2 + y**2} \\ \theta &= \tan\*\*{**-1**}(y / x) \end{aligned} \)$

参数#

  • y

    其类型应该是 real.

  • x

    其类型和种类参数应与y相同。如果y为零，则x必须不为零。

返回#

返回值具有与y相同的类型和种类参数。它是复数**(x+i，y)的主值。如果x不为零，则它位于-PI<=atan(X)<=PI的范围内。如果y为正，则返回符号为正。如果y为零，则如果x严格为正，则返回值为零；如果x为负且y为正零(或处理器不处理带符号零)，则PI为零；如果x为负且Y为负零，则-PI为负。最后，如果x为零，则结果的大小为PI/2**。

示例#

示例程序：

program demo_atan2
use,intrinsic :: iso_fortran_env, only : dp=>real64,sp=>real32
implicit none
real(kind=sp) :: x = 1.e0_sp, y = 0.5e0_sp, z
   z = atan2(y,x)
   write(*,*)x,y,z
end program demo_atan2

结果：

      1.00000000      0.500000000      0.463647604

Haversine 公式#

来自维基百科中关于“Haversine 公式”的文章：

The haversine formula is an equation important in navigation,
giving great-circle distances between two points on a sphere from
their longitudes and latitudes.

因此，要显示美国田纳西州纳什维尔国际机场 (BNA) 和美国加利福尼亚州洛杉矶国际机场 (LAX) 之间的大圆距离，你可以从它们的纬度和经度开始，通常为

BNA: N 36 degrees 7.2',   W 86 degrees 40.2'
LAX: N 33 degrees 56.4',  W 118 degrees 24.0'

转换为以度为单位的浮点值是：

     Latitude Longitude

   - BNA
     36.12, -86.67

   - LAX
     33.94, -118.40

然后使用半正弦公式粗略计算出沿地球表面的位置之间的距离：

示例程序：

program demo_sin
implicit none
real :: d
    d = haversine(36.12,-86.67, 33.94,-118.40) ! BNA to LAX
    print '(A,F9.4,A)', 'distance: ',d,' km'
contains
function haversine(latA,lonA,latB,lonB) result (dist)
!
! calculate great circle distance in kilometers
! given latitude and longitude in degrees
!
real,intent(in) :: latA,lonA,latB,lonB
real :: a,c,dist,delta_lat,delta_lon,lat1,lat2
real,parameter :: radius = 6371 ! mean earth radius in kilometers,
! recommended by the International Union of Geodesy and Geophysics

! generate constant pi/180
real, parameter :: deg_to_rad = atan(1.0)/45.0
   delta_lat = deg_to_rad*(latB-latA)
   delta_lon = deg_to_rad*(lonB-lonA)
   lat1 = deg_to_rad*(latA)
   lat2 = deg_to_rad*(latB)
   a = (sin(delta_lat/2))**2 + &
          & cos(lat1)*cos(lat2)*(sin(delta_lon/2))**2
   c = 2*asin(sqrt(a))
   dist = radius*c
end function haversine
end program demo_sin

结果：

    distance: 2886.4446 km

random_number#

名称#

random_number(3) - [MATHEMATICS:RANDOM] 伪随机数

语法#

   random_number(harvest)

说明#

从 0 <= x < 1 范围内的均匀分布中返回单个伪随机数或伪随机数数组。

参数#

  • harvest

    应为 real 类型的标量或数组。

示例#

示例程序：

program demo_random_number
use, intrinsic :: iso_fortran_env, only : dp=>real64
implicit none
integer, allocatable :: seed(:)
integer              :: n
integer              :: first,last
integer              :: i
integer              :: rand_int
integer,allocatable  :: count(:)
real(kind=dp)        :: rand_val
   call random_seed(size = n)
   allocate(seed(n))
   call random_seed(get=seed)
   first=1
   last=10
   allocate(count(last-first+1))
   ! To have a discrete uniform distribution on the integers
   ! [first, first+1, ..., last-1, last] carve the continuous
   ! distribution up into last+1-first equal sized chunks,
   ! mapping each chunk to an integer.
   !
   ! One way is:
   !   call random_number(rand_val)
   ! choose one from last-first+1 integers
   !   rand_int = first + FLOOR((last+1-first)*rand_val)
      count=0
      ! generate a lot of random integers from 1 to 10 and count them.
      ! with a large number of values you should get about the same
      ! number of each value
      do i=1,100000000
         call random_number(rand_val)
         rand_int=first+floor((last+1-first)*rand_val)
         if(rand_int.ge.first.and.rand_int.le.last)then
            count(rand_int)=count(rand_int)+1
         else
            write(*,*)rand_int,' is out of range'
         endif
      enddo
      write(*,'(i0,1x,i0)')(i,count(i),i=1,size(count))
end program demo_random_number

exp(3) - [MATHEMATICS] 指数函数

说明#

program demo_exp
implicit none
real :: x, re, im
complex :: cx

   x = 1.0
   write(*,*)"Euler's constant is approximately",exp(x)

   !! complex values
   ! given
   re=3.0
   im=4.0
   cx=cmplx(re,im)

   ! complex results from complex arguments are Related to Euler's formula
   write(*,*)'given the complex value ',cx
   write(*,*)'exp(x) is',exp(cx)
   write(*,*)'is the same as',exp(re)*cmplx(cos(im),sin(im),kind=kind(cx))

   ! exp(3) is the inverse function of log(3) so
   ! the real component of the input must be less than or equal to
   write(*,*)'maximum real component',log(huge(0.0))
   ! or for double precision
   write(*,*)'maximum doubleprecision component',log(huge(0.0d0))

   ! but since the imaginary component is passed to the cos(3) and sin(3)
   ! functions the imaginary component can be any real value

end program demo_exp

平方#

名称#

sqrt(3) - [MATHEMATICS] 平方根函数

语法#

result = sqrt(x)

   TYPE(kind=KIND) elemental function sqrt(x) result(value)
   TYPE(kind=KIND),intent(in) :: x
   TYPE(kind=KIND) :: value

其中 TYPE 可以是 real 或 complex 并且 KIND 可以是任何对声明类型有效的类型。

说明#

sqrt(x) 计算 x 的主平方根。

在数学中，数 x 的平方根是数 y 使得 y*y = x。

考虑其平方根的数字被称为 radicand（弧度） 。

例如，9 的主平方根是 3，即使 (-3)*(-3) 也是 9。

real radicand 必须是正数。

负数的平方根是复数的一种特殊情况，其中 radicand 的分量不必是正数才能获得有效的平方根。

参数#

  • x

    如果 x 是 real 它的值必须大于或等于零。类型应为 real 或 complex。

返回#

返回值的类型为 real 或 complex。 kind 类型参数同x。

bessel_j0#

名称#

bessel_j0(3) - [数学] 第一类0阶Bessel函数

语法#

    result = bessel_j0(x)

说明#

bessel_j0(x) 计算 x 的第一类 0 阶贝塞尔函数。

参数#

  • x

    其类型应该是 real.

返回#

返回值的类型为 real，取值范围 -0.4027 <= bessel(0,x) <= 1 。与 x 种类相同。

示例#

示例程序：

program demo_besj0
use, intrinsic :: iso_fortran_env, only : real_kinds, &
& real32, real64, real128
   implicit none
   real(kind=real64) :: x = 0.0_real64
   x = bessel_j0(x)
   write(*,*)x
end program demo_besj0

结果：

      1.0000000000000000

bessel_jn#

名称#

bessel_jn(3) - [MATHEMATICS] 第一类贝塞尔函数

语法#

  result = bessel_jn(n, x)

  result = bessel_jn(n1, n2, x)

说明#

bessel_jn(n, x) 计算 x 的第一类 n 的贝塞尔函数。如果 n 和 x 是数组，它们的维度和形状应该一致。

bessel_jn(n1, n2, x) 返回一个贝塞尔函数作用后的数组，第一类贝塞尔函数的阶数为 n1 到 n2。

参数#

  • n

    应为 _ integer_ 类型的标量或数组。

  • n1

    应为 integer 类型的非负标量。

  • n2

    应为 integer 类型的非负标量。

  • x

    应为标量或类型为 real 的数组。对于 ** bessel_jn(n1, n2,x)**，它应为标量。

返回#

返回值是 real 类型的标量。与 x 种类相同。

示例#

示例程序：

program demo_besjn
use, intrinsic :: iso_fortran_env, only : real_kinds, &
   & real32, real64, real128
implicit none
real(kind=real64) :: x = 1.0_real64
    x = bessel_jn(5,x)
    write(*,*)x
end program demo_besjn

结果：

      2.4975773021123450E-004

erfc#

名称#

erfc(3) - [MATHEMATICS] 互补误差函数

语法#

result = erfc(x)

   elemental function erfc(x)
   real(kind=KIND) :: erfc
   real(kind=KIND),intent(in) :: x

说明#

erfc(x) 计算 x 的互补误差函数。简单地说这等价于1 - erf(x)，但是提供erfc是因为如果erf(x)被调用大x会极大地损失相对精度并从 1 中减去结果。

erfc(x) 定义为

\[ \text{erfc}(x) = 1 - \text{erf}(x) = 1 - \frac{2}{\sqrt{\pi}} \int_x^{\infty} e^{-t^2} dt. \]

参数#

  • x

    其类型应该是 real.

返回#

返回值是 real 类型，与 x 类型相同。它位于范围内

0 <= erfc(x) <= 2.

示例#

示例程序：

program demo_erfc
use, intrinsic :: iso_fortran_env, only : &
 & real_kinds, real32, real64, real128
implicit none
real(kind=real64) :: x = 0.17_real64
    write(*,*)x, erfc(x)
end program demo_erfc

结果：

     0.17000000000000001       0.81000753879819121

gamma#

名称#

gamma(3) - [MATHEMATICS] Gamma 函数，产生正整数的阶乘

语法#

x = gamma(x)

说明#

gamma(x) 计算 x 的 Gamma。对于 n 的正整数值，Gamma 函数可用于计算阶乘，如 (n-1)! == gamma(real(n))。那是

n! == gamma(real(n+1))
\[\begin{split} \\__Gamma__(x) = \\int\_0\*\*\\infty t\*\*{x-1}{\\mathrm{e}}\*\*{__-t__}\\,{\\mathrm{d}}t \end{split}\]

参数#

  • x

    应为 real 类型，既不是零也不是负整数。

返回#

返回值的类型为 real，与 x 类型相同。

示例#

示例程序：

program demo_gamma
use, intrinsic :: iso_fortran_env, only : wp=>real64
implicit none
real :: x, xa(4)
integer :: i

   x = gamma(1.0)
   write(*,*)'gamma(1.0)=',x

   ! elemental
   xa=gamma([1.0,2.0,3.0,4.0])
   write(*,*)xa
   write(*,*)

   ! gamma(3) is related to the factorial function
   do i=1,20
      ! check value is not too big for default integer type
      if(factorial(i).gt.huge(0))then
         write(*,*)i,factorial(i)
      else
         write(*,*)i,factorial(i),int(factorial(i))
      endif
   enddo
   ! more factorials
   FAC: block
   integer,parameter :: n(*)=[0,1,5,11,170]
   integer :: j
      do j=1,size(n)
         write(*,'(*(g0,1x))')'factorial of', n(j),' is ', &
          & product([(real(i,kind=wp),i=1,n(j))]),  &
          & gamma(real(n(j)+1,kind=wp))
      enddo
   endblock FAC

   contains
   function factorial(i) result(f)
   integer,parameter :: dp=kind(0d0)
   integer,intent(in) :: i
   real :: f
      if(i.le.0)then
         write(*,'(*(g0))')'<ERROR> gamma(3) function value ',i,' <= 0'
         stop      '<STOP> bad value in gamma function'
      endif
      f=gamma(real(i+1))
   end function factorial
end program demo_gamma

norm2#

名称#

norm2(3) - [MATHEMATICS] 欧几里得向量范数

语法#

result = norm2(array, dim)

real function result norm2(array, dim)

   real,intent(in) :: array(..)
   integer,intent(in),optional :: dim

说明#

沿维度 dim 计算 array 的欧几里得向量范数（L_2 范数）。

参数#

  • 数组

    应为 real 类型的数组。

  • 暗淡

    应该是 integer 类型的标量，其值在 1 到 rank(array) 的范围内。

返回#

结果与 array 的类型相同。

如果 dim 不存在，则返回具有 array 元素的平方和的平方根的标量。

否则，将返回一个秩为 n-1 的数组，其中 n 等于 array 的秩，并返回一个与 array 相似的形状，其中维度 DIM 被删除。

示例#

示例程序：

program demo_norm2
implicit none

real :: x(3,3) = reshape([ &
   1, 2, 3, &
   4, 5, 6, &
   7, 8, 9  &
],shape(x),order=[2,1])

write(*,*) 'x='
write(*,'(4x,3f4.0)')transpose(x)

write(*,*) 'norm2(x)=',norm2(x)

write(*,*) 'x**2='
write(*,'(4x,3f4.0)')transpose(x**2)
write(*,*)'sqrt(sum(x**2))=',sqrt(sum(x**2))

end program demo_norm2
