merge#

名称#

merge(3) - [ARRAY CONSTRUCTION] 合并变量

语法#

result = merge(tsource, fsource, mask)

说明#

基本函数 merge(3) 根据逻辑掩码从两个数组或标量中选择值。结果等于 tsource 的元素，其中 mask 的对应元素是 .true.，或者 fsource 的元素它是.false。 .

支持多维数组。

请注意，merge(3) 的参数表达式不需要短路，因此（例如）如果数组 x 在标准下面的语句中包含零值，则防止产生浮点除以零；因为 1.0/x 可以在掩码用于选择要保留的值之前针对 x 的所有值进行评估：

      y = merge( 1.0/x, 0.0, x /= 0.0 )

请注意，编译器也可以自由短路或生成无穷大，因此这可能适用于许多编程环境，但不推荐使用。

对于像这样的情况，可以通过 where 构造使用掩码赋值：

      where(x .ne. 0.0)
         y = 1.0/x
      elsewhere
         y = 0.0
      endwhere

而不是更晦涩

      merge(1.0/merge(x,1.0,x /= 0.0), 0.0, x /= 0.0)

参数#

  • tsource

    可以是任何类型，包括用户定义的。

  • fsource

    应与 tsource 具有相同的类型和类型参数。

  • mask

    应为 logical 类型。

请注意，（当前）character 值必须具有相同的长度。

返回值#

结果与 tsource 的类型和类型参数相同。对于任何元素，如果 mask 为真，则结果为 tsource，否则为 fsource。

示例#

的价值

     merge (1.0, 0.0, k > 0)

K=5 为 1.0，K=-2 为 0.0。

program demo_merge
implicit none
integer :: tvals(2,3), fvals(2,3), answer(2,3)
logical :: mask(2,3)
integer :: i
logical :: chooseleft

   tvals(1,:)=[  10, -60,  50 ]
   tvals(2,:)=[ -20,  40, -60 ]

   fvals(1,:)=[ 0, 3, 2 ]
   fvals(2,:)=[ 7, 4, 8 ]

   mask(1,:)=[ .true.,  .false., .true. ]
   mask(2,:)=[ .false., .false., .true. ]

   write(*,*)'mask of logicals'
   answer=merge( tvals, fvals, mask )
   call printme()

   write(*, *)'highest values'
   answer=merge( tvals, fvals, tvals > fvals )
   call printme()

   write(*, *)'lowest values'
   answer=merge( tvals, fvals, tvals < fvals )
   call printme()

   write(*, *)'zero out negative values'
   answer=merge( tvals, 0, tvals < 0)
   call printme()

   write(*, *)'binary choice'
   chooseleft=.false.
   write(*, '(3i4)')merge([1,2,3],[10,20,30],chooseleft)
   chooseleft=.true.
   write(*, '(3i4)')merge([1,2,3],[10,20,30],chooseleft)

contains

subroutine printme()
      write(*, '(3i4)')(answer(i, :), i=1, size(answer, dim=1))
end subroutine printme

end program demo_merge

    zero out negative values

    -20   0 -60

pack#

名称#

pack(3) - [ARRAY CONSTRUCTION] 将一个数组打包成一个秩为 1 的数组

语法#

result = pack(array, mask,vector)

   TYPE(kind=KIND) function pack(array,mask,vector)
   TYPE(kind=KIND),option(in) :: array(*)
   logical  :: mask(*)
   TYPE(kind=KIND),option(in),optional :: vector(*)

其中 TYPE(kind=KIND) 可以是任何类型，其中 array 和 vector 和返回的值必须是相同的类型。 mask 可以是标量，也可以是数组。

说明#

将 ARRAY 的元素存储在一个一维数组中。

结果数组的开头由 **mask_**等于 .true. 的元素组成。之后，位置被从 vector中获取的元素填充。

参数#

  • 数组

    应是一个任意类型的数组。

  • mask

    应为 logical 类型的数组，大小与 array相同。或者，它可以是 logical 标量。

  • 向量

    （可选）应为与 array 类型相同且秩为 1 的数组。如果存在，vector中的元素数量应等于或大于** mask** 中真实元素的数量。如果 ** mask** 是标量，则** vector** 中的元素数应等于或大于** array**中的元素数。

返回值#

结果是一个秩为 1 的数组，与 ** array** 的类型相同。如果存在 ** vector** ，则结果大小为 ** vector** 的大小，否则为 ** mask** 中 ** .true.** 值的数量。

示例#

示例程序：

program demo_pack
implicit none
   call test1()
   call test2()
   call test3()
contains
!
subroutine test1()
! gathering nonzero elements from an array:
integer :: m(6)

   m = [ 1, 0, 0, 0, 5, 0 ]
   write(*, fmt="(*(i0, ' '))") pack(m, m /= 0)  ! "1 5"

end subroutine test1
!
subroutine test2()
! Gathering nonzero elements from an array and appending elements
! from VECTOR till the size of the mask array (or array size if the
! mask is scalar):
integer :: m(4)

   m = [ 1, 0, 0, 2 ]
   write(*, fmt="(*(i0, ' '))") pack(m, m /= 0, [ 0, 0, 3, 4 ])

end subroutine test2
!
subroutine test3()
! select strings whose second character is "a"
character(len=10) :: m(4)

m = [ character(len=10) :: 'ape', 'bat', 'cat', 'dog']
   write(*, fmt="(*(g0, ' '))") pack(m, m(:)(2:2) == 'a' )

end subroutine test3
!
end program demo_pack

结果：

   1 5
   1 2 3 4
   bat        cat

spread#

名称#

** spread** (3) - [ARRAY CONSTRUCTION] 向数组添加维度

语法#

result = spread(source, dim, ncopies)

  TYPE(kind=KIND) function spread(source, dim, ncopies)

   TYPE(kind=KIND)    :: source(..)
   integer,intent(in) :: dim
   integer,intent(in) :: ncopies

说明#

沿指定维度 ** dim** 复制 ** source** 数组 ** ncopies** 次。

如果 source 是标量，则结果的形状为 (MAX (NCOPIES, 0))。结果的每个元素都有一个等于 source 的值。

参数#

  • 资源

    应是一个标量或任意类型的小于15维的数组。

  • 暗淡

    应该是 integer 类型的标量，其值在 1 到 n+1 的范围内，其中 n 等于 source 的等级。

  • n份

    应是一个类型为 integer 的标量。

返回值#

结果是一个与 source 类型相同的数组，排名为 n+1，其中 n 等于 source 的排名。

示例#

示例程序：

program demo_spread
implicit none
integer :: a = 1, b(2) = [ 1, 2 ]

   write(*,*) spread(a, 1, 2)            ! "1 1"
   write(*,*) spread(b, 1, 2)            ! "1 1 2 2"

end program demo_spread

unpack#

名称#

unpack(3) —— [ARRAY CONSTRUCTION] 使用掩码将向量的元素分散到数组中

语法#

result = unpack(vector, mask, field)

 type(TYPE(kind=KIND)),intent(in) :: vector(:)
 logical,intent(in)               :: mask(..)
 type(TYPE(kind=KIND)),intent(in) :: field(..)
 type(TYPE(kind=KIND))            :: result(..)

说明#

按数组元素顺序使用 mask 中的 .true. 将vector的元素分散到任意维度数组field的中，以指定vector值的位置。

因此，将生成 field 的副本，并将所选择的元素替换为 vector 中的值。

参数#

  • 向量

    应为任何类型且排名第一的数组。 它至少应具有与 mask 具有 .true. 值一样多的元素。

  • mask

    应为 logical 类型的数组。

  • 场地

    应与vector具有相同的类型和类型参数，并应符合mask。

返回值#

结果是一个类型和类型参数与 vector 相同且形状与 mask 相同的数组。

按照数组元素顺序，对应于 MASK 的第 i 个 true 元素的结果元素具有 i = 1, 2, …, t，其中 t 是 MASK 中的真值数。如果 FIELD 是标量，则每个其它元素的值等于 FIELD，如果它是数组，则等于 FILED 的相应元素。

结果数组对应于 field，其中 mask 的 .true. 元素按数组元素顺序替换为 vector 中的值。

示例#

通过 unpack，可以将特定值“Scattered”到数组中的特定位置

                       1 0 0
    If M is the array  0 1 0
                       0 0 1

    V is the array [1, 2, 3],
                               . T .
    and Q is the logical mask  T . .
                               . . T
    where "T" represents true and "." represents false, then the result of

    UNPACK (V, MASK = Q, FIELD = M) has the value

      1 2 0
      1 1 0
      0 0 3

    and the result of UNPACK (V, MASK = Q, FIELD = 0) has the value

      0 2 0
      1 0 0
      0 0 3

示例程序：

program demo_unpack
implicit none
integer :: vector(2)  = [1,1]
logical :: mask(4)  = [ .true., .false., .false., .true. ]
integer :: field(2,2) = 0, unity(2,2)

   ! result: unity matrix
   unity = unpack(vector, reshape(mask, [2,2]), field)
   write(*,*)unity,size(unity),shape(unity)

end program demo_unpack

结果：

              1           0           0           1           4
              2           2

allocated#

名称#

allocated(3) - [ARRAY INQUIRY] 可分配实体的状态

语法#

  result = allocated(array)

或

  result = allocated(scalar)

说明#

allocated(array) 和 allocated(scalar) 分别检查 array 和 scalar 的分配状态。

参数#

  • 数组

    参数应为 allocatable 数组。

  • 标量

    参数应是一个 allocatable 标量。

返回值#

返回值是带有默认逻辑种类标量 logical 类型。如果分配了参数，则结果为 .true。；否则，它返回 .false..

示例#

示例程序：

program demo_allocated
use,intrinsic :: iso_fortran_env, only : dp=>real64,sp=>real32
implicit none
integer :: i = 4
real(kind=sp), allocatable :: x(:)

   ! if already allocated, deallocate
   if ( allocated(x) ) deallocate(x)

   ! only if not allocated, allocate
   if ( .not. allocated(x) ) allocate(x(i))

   write(*,*)allocated(x), size(x)
   if( allocated(x)) then
       write(*,*)'do things if allocated'
   else
       write(*,*)'do things if not allocated'
   endif
   call intentout(x)
   write(*,*)'note it is deallocated!',allocated(x)
   contains
   subroutine intentout(arr)
   ! note that if arr has intent(out) and is allocatable,
   ! arr is deallocated on entry
   real(kind=sp),intent(out),allocatable :: arr(:)
       write(*,*)'note it was allocated in calling program',allocated(arr)
   end subroutine intentout

end program demo_allocated

结果：

    T           4
    do things if allocated
    note it was allocated in calling program F
    note it is deallocated! F

is_contiguous#

名称#

is_contiguous(3) - [ARRAY INQUIRY] 测试对象是否连续

语法#

result = is_contiguous(a)

说明#

当且仅当对象是连续的时为真。

一个对象是连续的，如果它是

  • (1) 具有 CONTIGUOUS 属性的对象，

  • (2) 非假定形状的非指针整个数组，

  • (3) 一个假定形状的数组，它是与连续数组关联的参数，

  • (4) 由 ALLOCATE 语句分配的数组，

  • (5) 与连续目标关联的指针，或

  • (6) 一个非零大小的数组部分，前提是

    • (a) 它的基础对象是连续的，

    • (b) 它没有向量下标，

    • (c) 该部分的元素，按数组元素顺序，是按数组元素顺序连续的基本对象元素的子集，

    • (d) 如果数组是字符类型并且出现子字符串范围，则子字符串范围指定父字符串的所有字符，

    • (e) 只有其最后的部分引用具有非零等级，并且

    • (f) 它不是复数类型数组的实部或虚部。

如果对象是数组子对象，则对象不连续，并且

  • 该对象有两个或多个元素，

  • 数组元素顺序中对象的元素在基础对象的元素中不连续，

  • 该对象不是长度为零的字符类型，并且

  • 该对象不是派生类型，除了零大小的数组之外没有任何最终组件，并且

  • 字符的长度为零。

任何其它对象是否连续取决于处理器。

参数#

  • a

    可以是任何类型。它应该是一个数组。如果它是一个指针，它应该被关联。

返回值#

  • 结果

    类型默认逻辑标量。如果 a 是连续的，则结果的值为 true，否则为 false。

示例#

示例程序：

program demo_is_contiguous
implicit none
intrinsic is_contiguous
real, DIMENSION (1000, 1000), TARGET :: A
real, DIMENSION (:, :), POINTER       :: IN, OUT
   IN => A              ! Associate IN with target A
   OUT => A(1:1000:2,:) ! Associate OUT with subset of target A
   !
   write(*,*)'IN is ',IS_CONTIGUOUS(IN)
   write(*,*)'OUT is ',IS_CONTIGUOUS(OUT)
   !

      (s1, s2, . . ., sDIM-1, sDIM+1, . . ., sn )

名称#

all(3) - [数组归约] 确定所有值是否为真

语法#

result = all(mask, dim)

说明#

mask 的元素沿维度 dim 做逻辑与。

“all(mask, dim)” 确定沿维度 dim 的数组中 mask 中的值是否均为真。

参数#

  • mask

    应该是一个逻辑数组。也就是说，参数的类型应该是 logical 并且不能是标量。

  • 暗淡

    （可选）dim 应该是一个标量整数，其值介于 1 和 mask 的维度之间。相应的实参不应是具有可选属性的虚参。

返回值#

“all(mask)” 返回一个 logical 类型的标量值，其 kind 类型参数与 mask 的 kind 类型参数相同。如果存在 dim，则 all(mask, dim) 返回一个秩为 mask - 1 的数组。形状由 mask 决定，其中第 dim 维将被省略。

  1. 如果 mask 的所有元素都为真，则 all(mask) 为真。如果 mask 的大小为零，则也是如此；否则为假。

  2. 如果 mask 的维度为 1，则 all(mask, dim) 等价于 all(mask)。如果维度大于 1，则通过将 all() 应用于数组切片来确定 all(mask, dim)。

  4. 结果值。

    情况 （i）：

    如果 mask 的所有元素为真或 mask 的大小为零，则 all(mask) 的结果值为真，如果mask 的任意元素为假，则结果值为假。

    情况（ii）：

    如果 mask 的秩为 1，则 all(mask,dim) 等于 all(mask)。否则，所有 (mask, dim) 的元素 (s1, s2, …, sdim-1, sdim+1, …, sn ) 的值相等到 all(掩码 (s1, s2, … ., sdim-1, :, sdim+1, …, sn))。

product#

名称#

product(3) - [数组归约] 数组元素的乘积

语法#

  result = product(array, dim, mask)

    NUMERIC,intent(in) :: array(..)
    integer,intent(in),optional :: dim
    logical,intent(in),optional :: mask(..)

其中 NUMERIC 是任何数字类型

说明#

如果 mask 中的对应元素为 .true.，则将 array 的所有选定元素相乘，或沿维度 dim 相乘。

如果 dim 不存在，则返回一个标量，其中包含 array 中所有元素的乘积。 （注意零大小的 array 返回 1）。

当 dim 存在时，如果掩码数组的维度为 1（即是向量），则结果是标量。否则，维度为 n-1 的数组，其中 n 等于 array 的维度，形状类似于删除 array 的第dim 维。

参数#

  • 数组

    应为 integer、real 或 complex 类型的数组。

  • 暗淡

    应该是 integer 类型的标量，其值在 1 到 n 的范围内，其中 n 等于 array 的维度。

  • mask

    应为 logical 类型，并且可以是标量或与 array 形状相同的数组。

返回值#

结果与 array 的类型相同。

program demo_product

!    selected array elements.

      print all, 'factorial of ',n,' is ', product([(real(i),i=1,n)])
   enddo

   array=[10,12,13,15,20,25,30]
   p1=product(array, mask=mod(array, 2)==1) ! only odd elements
   p2=product(array, mask=mod(array, 2)/=1) ! only even elements
   print all, nl,'product of all elements',product(array) ! all elements
   print all, ' odd * even =',nl,p1,'*',p2,'=',p1*p2

   ! NOTE: If ARRAY is a zero-sized array, the result is equal to one
   print all
   print all, 'zero-sized array=>',product([integer :: ])
   ! NOTE: If nothing in the mask is true, this also results in a null
   !       array
   print all, 'all elements have a false mask=>',product(array,mask=.false.)

endblock NO_DIM

sum#

名称#

sum(3) - [数组归约] 对数组元素求和

语法#

   result = sum(array[, mask])
   result = sum(array, dim[, mask])

说明#

如果 MASK 中的对应元素为 TRUE，则沿维度 DIM 添加 ARRAY 的元素。

参数#

  • 数组

    应为 integer、real 或 complex 类型的数组。

  • 暗淡

    （可选）应该是一个 integer 类型的标量，其值在 1 到 n 的范围内，其中 n 等于 ARRAY 的维度。

  • mask

    （可选）应为 logical 类型，并且可以是标量或与 ARRAY 形状相同的数组。

返回值#

结果与 ARRAY 的类型相同。

如果 dim(3) 不存在，则返回一个标量，其中包含 ARRAY 中所有元素的总和。否则，将返回一个维度为 n-1 的数组，其中 n 等于 ARRAY 的维度，并返回一个与删除 ARRAY 第 DIM 维相似的形状。

示例#

示例程序：

program simple_sum
implicit none
integer :: x(5) = [ 1, 2, 3, 4, 5 ]
   print *, sum(x)                        ! all elements, sum = 15
   print *, sum(x, mask=mod(x, 2)==1)     ! odd elements, sum = 9
end program simple_sum

使用 MASK 选项演示 Fortran 90 SUM 函数

data b/ndh*-1.0, nduh*2.0/
   csum= sum(c(1:nd))
   cpsum= sum (c(1:nd), mask=c.gt.0)
   cbpsum= sum(c(1:nd), mask=b.gt.0.0)
   print *, 'Sum of all elements in c = ', csum
   print *, 'Sum of Positive elements in c = ', cpsum
   print *, 'Sum of elements in c when corresponding elements in b>0', &
   & ' =', cbpsum

reshape#

名称#

reshape(3) - [数组重塑] 重塑数组的函数

语法#

result = reshape(source, shape, pad, order)

说明#

重塑数组 source 以对应于 shape。如有必要，可以使用 pad 中的元素填充新数组，或者按照 order 的定义进行置换。

参数#

  • 资源

    任何类型的数组。

  • shape

    维度为 1 的数组，类型为 integer。它的值必须是正数或零。

  • pad

    （可选）与 source 类型相同的数组。

  • order

    （可选）类型为 integer 且形状与 shape 相同的数组。它的值应该是从 1 到 n 的数字置换，其中 n 是 shape 的大小。如果 order 不存在，则假定为自然排序。

返回值#

结果是一个形状为 shape 的数组，其类型与 source 相同。

示例#

示例程序：

program demo_reshape
implicit none
integer :: i
integer, dimension(4) :: x=[(i,i=10,40,10)]
real :: xx(3,4)
real,allocatable :: v(:)
    ! x is originally a vector with four elements
    write(*,*) shape(x) ! what is the current shape of the array?
    write(*,*) shape(reshape(x, [2, 2]))    ! prints "2 2"

    ! pack any array into a vector
    xx=1.0
    v=reshape(xx,[size(xx)])
    write(*,*)shape(v),ubound(v)
end program demo_reshape
